## AIM : 
To Design the Digital to Analog Converter circuit using LT-Spice


## SOFTWARE REQUIRED: 
LT-Spice


## SIMULATION OUTPUT:

## PROCEDURE:
•	Double click on LT-Spice icon.
•	New schematic window open.
•	Pick and paste the required component from the library and draw the circuit diagram .
•	Complete the connection.
•	Save the file by giving file name.
•	Click on the run option ->click advanced open ->select Ac analysis->enter the amplitude time delay stop time value.
•	Click on the run option ->simulation window opens->place the probe ->output graph is obtaine

## RESULT:
Thus the LT-SPICE Tool has been studied and Digital to Analog circuit is simulated.
